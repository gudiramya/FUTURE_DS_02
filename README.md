# Customer Retention & Churn Analysis Dashboard

## 📌 Project Overview
This Power BI dashboard analyzes customer retention and churn patterns using a customer dataset. It helps identify customer behavior, churn reasons, subscription trends, and key business metrics.

## 📊 Dashboard Features
- Total Customers
- Active Customers
- Churned Customers
- Churn Rate
- Retention Rate
- Total Revenue
- Customer Status Analysis
- Customer Distribution by Year
- Subscription Plan Analysis
- Churn by City
- Age Group Analysis
- Gender-wise Customer Status
- Monthly Fee Analysis
- Churn Reason Analysis

## 🛠️ Tools Used
- Power BI
- Microsoft Excel
- Power Query (Data Cleaning)
- DAX Measures

## 📂 Dataset
The dataset contains customer information, including:
- Customer ID
- Join Date
- Age Group
- Gender
- City
- Subscription Plan
- Monthly Fee
- Customer Status
- Churn Reason
- Revenue

## 📈 Key Insights
- Total Customers: **500**
- Active Customers: **418**
- Churned Customers: **82**
- Churn Rate: **16.40%**
- Total Revenue: **₹3M**
